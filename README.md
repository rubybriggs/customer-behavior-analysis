Project Title: Customer Shopping Behavior
1.  Overview
This project is a comprehensive data analysis and business intelligence solution designed to identify key drivers of sales performance and recommend strategic improvements].
The analysis spans the entire data pipeline, from data ingestion and cleaning using Python, to complex querying with SQL, culminating in an interactive dashboard built in Power BI for visual storytelling and actionable insights.

2.  Dataset
The core analysis uses the Customer Shopping Behavior dataset.
Key Metrics include Age, Gender/ Transaction_ID, Sale_Amount, Region, Product_Category, Order_Date]

3.  Tools & Technologies
This project utilizes a robust stack of industry-standard tools for data preparation, analysis, and visualization.
Programming: Python (for EDA, data cleaning, transformation)
Key Libraries: pandas
Database: MSSQL Server for running complex queries and data storage
Visualization: Power BI for interactive dashboard creation
Presentation: Microsoft Power Point for generating the final project presentation
